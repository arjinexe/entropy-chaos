# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.3.x   | ✅ Yes    |
| 0.2.x   | ⚠️ Best effort |
| < 0.2   | ❌ No     |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Email: security@yourdomain.com

We aim to respond within 48 hours and publish a fix within 14 days.
